RESTful API란?

REpresentational State Transfer
  • 핵심 3요소:
    1. 자원 (Resource): URI (/users)
    2. 행위 (Verb): Method (GET, POST)
    3. 표현 (Representation): JSON, XML

RESTful 디자인 예시

행위 나쁜 예 (Not RESTful) 좋은 예 (RESTful)
조회 POST /getUser GET /users/1
생성 GET /createUser POST /users
수정 POST /updateUser PUT /users/1
삭제 GET /deleteUser DELETE /users/1
